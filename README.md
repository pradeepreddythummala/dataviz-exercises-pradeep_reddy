# ⚽ FIFA World Cup Match Analysis Dashboard

This project analyzes the FIFA Men's World Cup Match Dataset (1970–2022) using Python, Plotly, and Streamlit. It provides interactive visualizations and insights into World Cup match statistics.

## 🚀 Live Dashboard
https://dataviz-exercises-pradeepreddy-wfyxgvgfqn3rqcrucni9pb.streamlit.app/

## 📂 GitHub Repository
https://github.com/pradeepreddythummala/dataviz-exercises-pradeepreddy

## 📊 Features
- Interactive Team Filter
- Dataset Preview
- Project Summary KPIs
- Top Teams by Win Percentage
- Average Goals by Tournament
- Possession vs Goals Analysis
- Goals Distribution by Stage
- Top Teams by Average Shots on Target
- Top Teams by Average Yellow Cards
- Highest Scoring Stadiums
- Corners vs Goals Analysis
- Average Possession by Team
- Correlation Heatmap

## 🛠 Technologies Used
- Python
- Pandas
- Plotly
- Streamlit
